# Swift Calculator App

A simple, user-friendly calculator application built with Swift and UIKit for iOS.

## Overview

This calculator app provides basic arithmetic operations with a clean interface. It displays both the working expression and the calculated result, making it easy to track your calculations.


## Features
 - **Basic Arithmatic Operations**
   - Addition(+)
   - Subtraction(-)
   - Multiplication(*)
   - Division(/)

- **Input Validation**
  - Prevents invalid expressions (operators at start/end)
  - Detects consecutive operators
  - Shows alert for invalid input
  
- **Additional Functions**
  - Percentage Calculations(%)
  - Decimal Point support(.)
  - Clear ALL
 
## Requirements
- IOS 13.0+
- XCode 11.0+
- Swift 5.0+
